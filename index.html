JavaScript Roadmap

O JavaScript é uma das linguagens mais populares do mundo, criada em 1995 por Brendan Eich. Inicialmente chamada de Mocha e depois LiveScript, ganhou o nome JavaScript para aproveitar a popularidade do Java. Sua proposta é trazer dinamismo e interatividade às páginas web.

Desde então, JavaScript evoluiu: Node.js permite rodar JS no servidor, e o padrão ECMAScript 6 (ES6) trouxe recursos modernos como let/const, arrow functions, classes e modules. Frameworks e bibliotecas como React, Angular e Vue consolidaram JavaScript no desenvolvimento moderno.

Java Roadmap

O Java é uma das linguagens mais utilizadas do mundo, criada em 1991 no Projeto Green da Sun Microsystems e lançada oficialmente em 1995. Voltada a aplicações Web, Mobile, Desktop e Backend, Java evoluiu com recursos como Generics, Metadata, Project Jigsaw e atualmente mantém versões LTS modernas.
